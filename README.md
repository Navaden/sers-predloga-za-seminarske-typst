# sers-predloga-za-seminarske-typst

## Kaj vsebuje ta predloga
napiši kaj vse je noter, Matic iz prihodnosti :)

## Namestitev
1. namesti typst (ali uporabi spletno različico)
2. Kloniraj repository
3. daj vse tvoje potrebne stvari (npr. slike) v eno mapo.
4. odpri terminal na lokaciji tvojega typst projekta in napiši "typst compile [ime_datoteke.typ]" ali "typst watch [ime_datoteke.typ]"
   - razlika med compile in watch:
     - compile datoteko compila skupaj v pdf samo enkrat -- ko je ukaz zagnan
     - watch sproti gleda spremembe (typst datoteko je treba sproti shranjevati) in posodablja končno datotekeo.

## Kako uporabljati
Najboljše deluje, ko imaš typst nastavljen na "watch" način in končno datoteko odprto v pdf viewerju tvoje izbire (takem, ki lahko v živo posodablja spremembe datoteke).
